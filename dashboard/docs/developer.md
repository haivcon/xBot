# Developer & Open Source

Hệ sinh thái **xLayer.my** tôn trọng cộng đồng mã nguồn mở. Chúng tôi chia sẻ mã nguồn cho các phần mềm liên quan đến an toàn tài sản của người dùng để mọi chuyên gia bảo mật đều có thể Audit (Kiểm toán).

## Mã nguồn mở

### Dự án xKey (Offline Vault)
- **Mã nguồn**: [github.com/haivcon/xkey](https://github.com/haivcon/xkey)
- **Công nghệ**: React, Vite, Tailwind CSS v4, CapacitorJS.
- Trọng tâm phát triển: UI/UX cho thiết bị di động, tối ưu hóa thuật toán mã hóa dữ liệu cục bộ.

### Dự án xBot Dashboard
- **Mã nguồn**: [github.com/haivcon/xbot](https://github.com/haivcon/xbot)
- **Công nghệ**: React, Vite.
- Trọng tâm phát triển: Giao diện web quản lý bot, kết nối với Telegram Mini App.

## Đóng góp (Contributing)

Chúng tôi hoan nghênh các lập trình viên trên toàn cầu tham gia đóng góp mã nguồn (Pull Requests) hoặc báo cáo lỗi (Issues) trên Github.

1. **Fork** repo tương ứng trên Github.
2. Tạo **Branch** tính năng mới (`git checkout -b feature/tinh-nang-hay`).
3. Commit thay đổi (`git commit -m 'Thêm tính năng hay'`).
4. Push lên branch (`git push origin feature/tinh-nang-hay`).
5. Mở một **Pull Request**.
